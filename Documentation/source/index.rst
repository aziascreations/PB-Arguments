Common Attributes
=================
| These constants are shared by all the different modules.
| However, they are not guaranteed to have the same value.

#Version_Major
^^^^^^^^^^^^^^
    Contains an integer representing the module's *major* version number.

#Version_Minor
^^^^^^^^^^^^^^
    Contains an integer representing the module's *minor* version number.

#Version_Patch
^^^^^^^^^^^^^^
    Contains an integer representing the module's *patch* version number.

#Version_Label$
^^^^^^^^^^^^^^^
    Contains a string representing the module's version number's label.

#Version$
^^^^^^^^^
    Contains a string representing the module's version formatted under the *Semantic Versioning 2.0.0* standard.
